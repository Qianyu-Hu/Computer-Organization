## Serial Communication
One single data line; Bits are sent over the line one by one; No dedicated lines for control signals
![Pasted image 20260502153939.png](assets/Pasted%20image%2020260502153939.png)
1.Data transfer rate
baud rate: symbol per second
bit rate: baud rate×log2N bit per second （N表示symbol中的状态）
2.Synchronization Methods
- **异步串行通信**：
    - 一次传输一个字节
    - 每个字节的开始是异步的，因此每个字节都需要通过**起始位**来实现发送方与接收方之间的同步。
- **同步串行通信**：
    - 一次传输一个数据块
    - 发送方和接收方在数据传输开始时会通过**同步字符**进行同步。

## 异步传输
### transmit line上的基本单元——character（byte）
![Pasted image 20260502160614.png](assets/Pasted%20image%2020260502160614.png)
1. **起始位**（Start）：一位低电平，表示一个字符开始传输。解决接收器时钟（RxC） 与输入信号（RxD）之间不同步
2. **7位或8位字符数据**（7/8-bit character）+**校验位**（Parity Bit）：从 最低有效位（lsb） 开始发送，到 最高有效位（msb）结束。
3. **停止位**（Stop bits）：1、1.5 或 2 位 的高电平，确保在下一个字符开始时有一个 下降沿（从空闲高电平变为低电平
字节之间的线路状态是空闲（高电平）

## 同步校验
### Transmission
1. Character-oriented Synchronous Transmission
以character为基本单元，用特殊的同步字符（SYN,STX）让接收方找到帧的开始

2. Bit-oriented Synchronous Transmission
![Pasted image 20260502200932.png](assets/Pasted%20image%2020260502200932.png)
以bit为基本单元，用唯一的Flag01111110标志帧的开始和结束
为了使这个序列不会出现在数列中间，
![Pasted image 20260502200821.png](assets/Pasted%20image%2020260502200821.png)

| --- | ------------------ | ----------------------- |
| 接收端 | Zero Bit Deletion  | 每遇到5个连续的1后面跟一个0，就删除这个 0 |

### Transmit & Receive
8-bit element之间是contiguous(连续的)，无delay，效率高
![Pasted image 20260502191856.png](assets/Pasted%20image%2020260502191856.png)
Transmit clock和receive clock的频率一致

![Pasted image 20260502194512.png](assets/Pasted%20image%2020260502194512.png)

| 部分                      |                                      |
| ----------------------- | ------------------------------------ |
| Transmitter Clock (TxC) | 频率由人/协议指定                            |
| 线路上的波形                  | 根据 TxC 和曼彻斯特规则生成                     |
| Extracted clock (RxC)   | 由信号中数据跳变的位置决定，告诉 Receiver 采样电路读下一位数据 |
| Received Data           | 在extracted clock上升沿读取                |
Manchester规则：在每个时钟中期的中间进行一次信号跳变，1的时候从低到高，0的时候从高到低（从而决定时钟周期首/末是否需要再进行一次信号的跳变）
